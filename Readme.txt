# 🎓 Student Result Management System (Excel)

## 📌 Project Overview

The **Student Result Management System** is an Excel-based project developed to manage student records efficiently. It demonstrates the use of fundamental Excel functions and features to automate result calculation, grade assignment, and student record searching.

This project was created as part of my Excel learning journey and covers concepts from basic to intermediate Excel.

---

## ✨ Features

- 📋 Student database management
- ✅ Automatic Pass/Fail calculation using IF function
- 🏅 Automatic Grade calculation using Nested IF
- 🔍 Student search using XLOOKUP
- 📑 Data Validation (Drop-down lists)
- 🎨 Conditional Formatting
- ↕️ Sorting student records
- 🔎 Filtering student records

---

## 🛠 Excel Concepts Used

- Text Functions
- Date & Number Functions
- IF Function
- Nested IF Function
- XLOOKUP
- Data Validation
- Conditional Formatting
- Sorting
- Filters

## 🚀 How to Use

1. Download the Excel workbook.
2. Open it using Microsoft Excel.
3. Browse the student records.
4. Enter a Roll Number in the search section.
5. View the student's details automatically.
6. Use sorting and filters to analyze the data.
