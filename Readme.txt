# Student Result Management System (Excel)
Project Overview

This project is an Excel-based Student Result Management System developed using core Excel functions. It allows users to manage student records, calculate results and grades automatically, search students by Roll Number, and organize data efficiently.

Features

- Student database management
- Automatic Pass/Fail calculation using IF
- Automatic Grade calculation using Nested IF
- Student search using XLOOKUP
- Data Validation drop-down lists
- Conditional Formatting
- Sorting and Filtering

Excel Functions Used

- IF
- Nested IF
- XLOOKUP
- Data Validation
- Conditional Formatting
- Sorting
- Filters

Project Files

- Student_Result_Management_System.xlsx
