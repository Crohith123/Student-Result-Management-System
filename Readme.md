# 🎓 Student Result Management System (Excel)

## 📌 Project Overview

The **Student Result Management System** is an Excel-based project developed to efficiently manage student records and automate result processing. The project demonstrates the use of essential Excel functions and features to calculate results, assign grades, validate data, search records, and organize student information.

This project was created as part of my Excel learning journey and showcases practical applications of Excel in educational data management.

---

## ✨ Features

- 📋 Student record management
- ✅ Automatic Pass/Fail calculation using IF function
- 🏆 Automatic Grade calculation using Nested IF
- 🔍 Student search using XLOOKUP
- 📑 Data Validation (Drop-down Lists)
- 🎨 Conditional Formatting to highlight failed students
- ↕️ Sorting student records by marks
- 🔎 Filtering records by Class, Gender, Result, and Grade

## 🚀 How to Use

1. Download or clone this repository.
2. Open **Student_Result_Management_System.xlsx** in Microsoft Excel.
3. Browse the student records.
4. Enter a Roll Number in the search section.
5. View the student's details automatically.
6. Use sorting and filters to analyze the records.
